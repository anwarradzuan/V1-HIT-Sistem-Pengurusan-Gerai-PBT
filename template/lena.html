Sidebar

Lena has a customized, fully responsibile .lena-sidebar for our two main layout types. The sidebar can be toggled by clicking on a .sidebar-toggle-item.

Markup
                    
                        <nav id="lena-sidebar" class="sidebar lena-sidebar lena-light lena-full open">
                            <div class="sidebar-sticky">
                                <ul class="nav flex-column">
                                    <li>
                                        <a class="nav-link" href="#">
                                            <span class="typcn typcn-chart-bar-outline"></span>
                                            <span class="content">Analytics </span>
                                        </a>
                                        </li>
                                    <li>
                               </div>
                        </nav>
Classes & Elements
#lena-sidebar Represents the global lena-sidebar. Triggers the navbar initialization.
.lena-sidebar Global lena sidebar classes
.open The sidebar is opened
.lena-* The layout mode (full/horizontal/normal)
.lena-* The navbar accepts two color schemes (light/dark)

Header

Lena has a customized, fully responsibile .lena-navbar for our two main layout types.

Markup
<nav class="navbar lena-navbar lena-full fixed-top  open"></nav>
Classes

.open The navbar is in open state (only for full and for normal layout modes)

Maps

Lena supports 3 types of map engines: Google Maps, Mapbox and Leafet. Except Leaflet, you need to have an API keys to use these mapping solutions. Obtain API key for Google Maps and ACCESS TOKEN for Mapbox.

  • Official GoogleMaps documentation: DOCUMENTATION
  • Official Mapbox documentation: DOCUMENTATION
  • Official Leaflet documentation: DOCUMENTATION

Markup (HTML)
<div id="mapId"></div>
Markup - GoogleMaps (JS)
new googleMaps.Map(container, options);
Markup - Mapbox (JS)
new mapboxgl.Map({container: container});
Markup - Leflet (JS)
L.map(container).setView(latLng, zoom);
Classes
You need to set map dimensions (width & height) in order to render it correctly.

Charts

Chart plugins included in Lena: Charts.js, Google Charts and Sparkline.

Charts.js is a very lightweight, simple and flexible JavaScript charting library for designers & developers. Official documentation can be found here.

Markup (HTML - Chart.js)
<canvas id="domId"></canvas>
Markup (HTML - Chart.js)
new Chart(ctx, {type: 'line', data: {labels: [], datasets: [] }});

Google Charts is a javascript-based powerful, simple, easy-to-use, powerful charting library. Official documentation can be found here.

Markup (HTML - Google Charts)
<div id="domId"></div>
Markup (HTML - Google Charts)

                    var chart = new google.visualization.PieChart(document.getElementById(domId));
                    chart.draw(data, options);

Sparkline.js generates sparklines (small inline charts) directly in the browser using data supplied either inline in the HTML, or via javascript. Official documentation can be found here.

Markup (HTML - Sparkline)
<div id="domId"></div>
Markup (HTML - Sparkline)
$(element).sparkline([1,2,3,4], { type: 'line' });

Chat

Chat sidebar template can be used for implementing various type of chat features.
Chat is implemented in .lena-chat-overlay DOM element which can be toggled by clickin on elements with [data-chat-toggle="toggle"] attribute.

Markup (HTML)
                    
                        <div class="lena-chat-overlay">
                            <div class="header"></div>
                            <div class="body">
                                <div class="lena-chat-element me">
                                    <div class="message">Hi there! How can I help you?</div>
                                </div>
                                <div class="lena-chat-element you">
                                    <div class="w-100 lena-display-inline-block">
                                        <img src="assets/img/avatar1.png" width="30" height="30" class="circle-shape outlined float-left shadow” alt="">
                                        <div class="message">Hey there! How it's going?</div>
                                    </div>
                                </div>
                                 </div>
                            <div class="footer”></div>
                        </div>
Classes
.lena-chat-overlay Represents the chat sidebar wrapper
.lena-chat-overlay .header Chat sidebar header container
.lena-chat-overlay .footer Chat sidebar footer container
.lena-chat-element Represents the actual message wrapper
.lena-chat-element.message Represents the actual message in the message wrapper
.lena-chat-element.me Message from me
.lena-chat-element.you Message from the other user

Search overlay

Lena provides a global search overlay element which can be toggled by clicking on a [data-search-toggle="toggle"} attributed element.

Markup (HTML)
                    
                        <div class="lena-search-overlay">
                            <input type="text" placeholder="Search here">
                            <div class="row mt-5 lena-results"></div>
                        </div>
Classes
.lena-search-overlay Represents the global search overlay wrapper
.lena-results Represents the results of the search
